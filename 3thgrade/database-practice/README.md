# database-practice

Индексы в базах данных (БД) — это структуры данных, используемые для ускорения операций поиска и извлечения информации из таблиц. Они позволяют эффективно находить строки в таблицах по определенным полям или комбинациям полей.

Селективностью индекса назвается характеристика, определяющая, насколько эффективно индекс разбивает данные.

Селективность индексов считается как 1 делить на количество уникальных значений в индексированном столбце (или столбцах).

Примеры селективности:

- Процент записей, получаемый по значению
